# Chorus — Architecture & Specifications

Chorus is the meta-repository for the Loophole DAW ecosystem.
It contains:

- High-level architectural documents
- IPC protocol specifications
- Realtime guidelines for engine development
- Task briefs for Cursor/Codex agents
- Cross-repo decision records (ADR)

No code is stored here — only the formal definition of how the system fits
together.
